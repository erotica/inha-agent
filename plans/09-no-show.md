# 외래 No-Show 예측 및 예약 최적화

> 부서: 외래/진료지원팀

---

## 🎯 목표

환자 특성과 과거 패턴으로 No-Show 위험을 예측하여 예약 시스템을 최적화하고 외래 효율을 높인다.

## 📊 필요 데이터

환자 예약 이력(No-Show/취소/방문), 환자 인구통계(나이/거주지/보험), 기상 조건/거리, 과거 No-Show 패턴, 진료과/의사별 편차

## 🛠 기술 접근법

**모델:** XGBoost + 오버부킹 최적화

**주요 Feature:**
- 과거 No-Show 횟수/비율
- 예약-진료 간격(일)
- 거주지-병원 거리
- 강수량/미세먼지
- 예약 시간대
- 진료과
- 보험 유형
- 연령대

## 📤 출력물

- 개인별 No-Show 확률
- 오버부킹 추천 인원
- 알림 문자/카톡 발송 추천 시간
- No-Show 고위험 환자 리스트

## 💰 기대 효과

No-Show율 30%↓, 외래 가동률 15%↑

## 📅 구현 계획

1주: Feature Engineering / 1주: XGBoost 모델 / 1주: 오버부킹 최적화 / 1주: 예약 시스템 연동
