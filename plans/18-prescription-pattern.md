# 처방 패턴 분석 및 진료 행태 피드백

> 부서: QA/진료지원팀

---

## 🎯 목표

의사별 처방 패턴을 분석하여 진료 행태 피드백을 제공하고 적정 처방을 유도한다.

## 📊 필요 데이터

의사별 처방 내역(약품/검사/영상), 진단-처방 매핑, 진료 가이드라인, 동료 그룹 평균 처방 패턴, 환자 진료 결과

## 🛠 기술 접근법

**모델:** Clustering(K-means) + SHAP + Peer Comparison

**주요 Feature:**
- 처방 빈도/강도
- 항생제 처방률
- 주사제 비율
- 검사 건수/환자
- 영상 검사 건수
- 고가 약제 처방 비율

## 📤 출력물

- 의사별 처방 프로파일 리포트
- 동료 그룹 대비 비교 차트
- 이상 처방 패턴 경고
- 가이드라인 기반 개선 추천

## 💰 기대 효과

적정 처방률 15%↑, 항생제 처방 감소

## 📅 구현 계획

1주: 데이터 집계 / 1주: Clustering + SHAP / 1주: 피드백 리포트 템플릿 / 1주: 대시보드
