# 의료 영상 판독 대기 시간 예측

> 부서: 영상의학과

---

## 🎯 목표

검사 접수 데이터를 기반으로 판독 대기 시간을 예측하고, 우선순위 기반 스케줄링으로 대기 시간을 단축한다.

## 📊 필요 데이터

검사 접수 로그(검사 종류/시간/긴급도), 판독사별 처리량/처리 속도, 판독 완료 시간 이력, 검사실별 가동률

## 🛠 기술 접근법

**모델:** XGBoost + Queue Simulation

**주요 Feature:**
- 검사 종류(X-ray/CT/MRI/US)
- 긴급도(Routine/Urgent/STAT)
- 판독사 수
- 시간대/요일
- 접수 누적량
- 판독사별 평균 처리 속도

## 📤 출력물

- 예상 판독 대기 시간
- 우선순위 기반 판독 순서 추천
- 지연 예측 알림
- 판독사 배정 추천

## 💰 기대 효과

판독 대기 시간 25%↓, 긴급 검사 처리 시간 40%↓

## 📅 구현 계획

1주: 데이터 수집 + Feature / 1주: XGBoost 대기시간 예측 / 1주: 큐 시뮬레이션 / 1주: 대시보드
