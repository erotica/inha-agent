# 간호사 근무표 자동 생성 및 최적화

> 부서: 간호부

---

## 🎯 목표

환자 중증도/간호 강도와 간호사 선호도를 반영한 최적 근무표를 자동 생성한다.

## 📊 필요 데이터

간호사 프로필(경력/직급/인증/선호 근무), 병동 환자 중증도/간호 강도, 법적 근무 기준(연속 근무/휴식), 과거 근무표, 휴가/교육 일정

## 🛠 기술 접근법

**모델:** OR(혼합 정수 최적화) + XGBoost(간호 강도 예측)

**주요 Feature:**
- 환자 중증도 점수
- 병상 가동률
- 간호사 대 환자 비율
- 법적 제약 조건
- 간호사 선호도(가중치)
- 야간/주말/공휴일 가중치

## 📤 출력물

- 최적 간호사 근무표(1개월)
- Shift별 필요 인원 추천
- 간호 강도 기반 추가 배치 알림
- 간호사 만족도 예측

## 💰 기대 효과

근무표 수립 시간 90%↓, 간호사 만족도 20%↑

## 📅 구현 계획

2주: 제약 조건 정의 / 2주: OR 최적화 모델 / 1주: 간호 강도 예측 / 1주: 웹 기반 근무표 편집기
