# 임상시험 환자 모집 자동 매칭

> 부서: 임상시험센터/IRB

---

## 🎯 목표

EMR 데이터를 분석하여 임상시험 조건에 부합하는 환자를 자동으로 선별 및 매칭한다.

## 📊 필요 데이터

EMR 환자 데이터(진단/병기/검사/유전자), 임상시험 프로토콜(선정/제외 기준), 과거 스크리닝 결과, IRB 승인 현황, 시험별 모집 진행률

## 🛠 기술 접근법

**모델:** NLP(프로토콜 파싱) + Rule-based Matching + Ranking

**주요 Feature:**
- 진단 코드 일치도
- 병기/중증도
- 연령/성별
- 과거 치료 이력
- 검사 결과(적격 여부)
- 제외 기준 매칭 여부

## 📤 출력물

- 시험별 적격 환자 리스트(우선순위)
- 환자별 매칭 가능 시험 목록
- 스크리닝 성공 확률 예측
- 모집 진행률 대시보드

## 💰 기대 효과

환자 모집 기간 40%↓, 스크리닝 효율 50%↑

## 📅 구현 계획

2주: NLP 프로토콜 파싱 / 2주: Rule Engine 매칭 / 1주: Ranking + 예측 / 1주: 시스템 통합
