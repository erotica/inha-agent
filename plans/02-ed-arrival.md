# 응급실 환자 도착 예측 및 의료진 배치

> 부서: 응급의학과

---

## 🎯 목표

응급실 환자 도착 패턴을 시간대별로 예측하여 적정 의료진을 배치하고 대기 시간을 단축한다.

## 📊 필요 데이터

과거 응급실 내원 기록(시간대/요일/중증도/진료과), 119 구급대 이송 데이터, 지역 인구 통계, 계절/독감/코로나19 유행 데이터, 대형 행사/사고 발생 정보

## 🛠 기술 접근법

**모델:** LSTM(시계열) + Prophet(계절성) + Queueing Theory(대기열)

**주요 Feature:**
- 시간대별 도착 분포
- 요일 효과
- 계절성
- 공휴일
- 특수일(명절/선거/대회)
- 기온/날씨
- 독감 지수

## 📤 출력물

- 시간대별 예측 도착 환자 수(중증도별)
- 필요 의사/간호사 수 추천
- 대기 시간 예측
- 약품/소모품 사전 준비 알림

## 💰 기대 효과

응급실 대기 시간 30%↓, 적정 의료진 배치율 20%↑

## 📅 구현 계획

2주: 데이터 수집 + Feature Engineering / 2주: LSTM + Prophet 모델 / 1주: 인력 최적화 알고리즘 / 1주: 실시간 대시보드
