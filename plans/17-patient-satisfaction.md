# 환자 만족도 예측 및 서비스 개선

> 부서: QA/고객만족팀

---

## 🎯 목표

EMR/설문 데이터로 환자 만족도를 예측하고 부서별 개선 포인트를 도출한다.

## 📊 필요 데이터

환자 만족도 설문 결과(과거), 진료 대기 시간, 의사/간호사 응대 시간, 병원 환경 평가, 재방문율

## 🛠 기술 접근법

**모델:** NLP + XGBoost + SHAP

**주요 Feature:**
- 대기 시간(접수→진료)
- 진료 시간
- 의사 커뮤니케이션 평가
- 간호사 응대
- 병동 환경
- 식사 만족도

## 📤 출력물

- 부서별/항목별 예상 만족도 점수
- 만족도 결정 요인 분석(SHAP)
- 개선 우선순위 추천
- 저만족 예측 환자 사전 관리 알림

## 💰 기대 효과

환자 만족도 10%↑, 부서별 맞춤 개선 가능

## 📅 구현 계획

1주: 데이터 통합 / 2주: NLP + XGBoost 모델 / 1주: SHAP 분석 / 1주: 대시보드
