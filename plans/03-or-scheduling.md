# 수술실 스케줄 최적화 및 가동률 예측

> 부서: 수술실/마취과

---

## 🎯 목표

수술 예약 데이터를 기반으로 수술실 사용 시간을 예측하고 최적 스케줄을 생성하여 수술실 가동률을 극대화한다.

## 📊 필요 데이터

과거 수술 건별 소요 시간(마취+수술+회복), 수술실별/진료과별 사용률, 응급 수술 발생 패턴, 의사별 수술 시간 프로파일, 수술 재고 현황

## 🛠 기술 접근법

**모델:** XGBoost Regressor(수술 시간 예측) + Constraint Programming(스케줄링)

**주요 Feature:**
- 수술 코드
- 집도의 ID
- 마취 종류
- 환자 나이/BMI/ASA score
- 진료과
- 수술실 유형
- 시간대

## 📤 출력물

- 수술 건별 예상 소요 시간
- 일별/주간 최적 수술실 배정표
- 응급 수술 여유 시간 확보 알림
- 지연 예측 및 사전 조정

## 💰 기대 효과

수술실 가동률 20%↑, 초과 근무 시간 15%↓

## 📅 구현 계획

2주: 수술 데이터 수집 및 Feature / 2주: XGBoost 수술시간 예측 / 2주: 스케줄링 최적화 / 1주: 대시보드
