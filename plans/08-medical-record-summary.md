# 의무 기록 자동 요약 및 코드 추천

> 부서: 의무기록실/QA

---

## 🎯 목표

LLM/NLP로 의사 진료 기록을 자동 요약하고 진단/수술 코드(KCD/EDI)를 추천한다.

## 📊 필요 데이터

의사 진료 기록 텍스트(SOAP 노트/퇴원 요약), 진단 코드(KCD-8) 매핑 데이터, 수술 코드(EDI) 데이터, 과거 코드 부여 이력

## 🛠 기술 접근법

**모델:** LLM(Claude Code / Llama) + NER + Classification

**주요 Feature:**
- 진료 기록 텍스트
- 진단명/시술명 추출
- 코드-진단 매핑 확률
- 유사 사례 코드 부여 이력

## 📤 출력물

- 자동 생성 진료 기록 요약
- 추천 진단 코드(확률 순위)
- 추천 수술 코드
- 코드 부여 근거 설명
- 누락 코드 경고

## 💰 기대 효과

의무 기록 작성 시간 50%↓, 코드 정확도 20%↑

## 📅 구현 계획

2주: LLM 파인튜닝/프롬프트 설계 / 2주: NER 진단/시술명 추출 / 1주: 코드 추천 엔진 / 1주: EMR 연동
