# 의료장비 예측 유지보수 (Predictive Maintenance)

> 부서: 의공학팀

---

## 🎯 목표

의료장비 작동 데이터를 분석하여 고장을 사전 예측하고 계획적 유지보수를 수행한다.

## 📊 필요 데이터

장비 로그(가동 시간/온도/진동/에러 코드), 유지보수 이력, 장비 스펙/수명, 사용 패턴(시간당 사용 횟수), 부품 교체 이력

## 🛠 기술 접근법

**모델:** LSTM(시계열) + Random Forest(고장 분류)

**주요 Feature:**
- 장비 온도/진동 추이
- 에러 코드 발생 빈도
- 사용 시간 누적
- 마지막 정비 후 경과일
- 부품 교체 주기 대비 사용량

## 📤 출력물

- 고장 예측 일시(7일 전 경보)
- 잔여 수명 예측
- 필요 부품/정비 항목 추천
- 장비별 유지보수 일정표

## 💰 기대 효과

장비 다운타임 50%↓, 유지보수 비용 25%↓

## 📅 구현 계획

2주: 장비별 센서/로그 데이터 수집 / 2주: LSTM 고장 예측 / 1주: 잔여 수명 모델 / 1주: 알림 시스템
