# 재입원 위험 예측 및 퇴원 관리

> 부서: 전 부서/QA

---

## 🎯 목표

퇴원 환자의 30일 내 재입원 위험을 예측하여 집중 관리 대상을 선별하고 재입원율을 낮춘다.

## 📊 필요 데이터

입원/퇴원 이력, 진단/수술/처방 정보, 검사 결과(Lab), 간호 평가, 사회경제적 요소(거주지/보호자 유무), 외래 추적 관찰 데이터

## 🛠 기술 접근법

**모델:** XGBoost + LACE Index + Survival Analysis

**주요 Feature:**
- LACE Index(LOS/Acute/Charlson/ED)
- 주 진단/동반질환
- 퇴원 전 Lab 마지막 값
- 퇴원처(집/요양원)
- 약제 수
- 정기 외래 방문 여부

## 📤 출력물

- 재입원 위험 점수(LOW/MED/HIGH)
- 위험 요인별 설명
- 퇴원 후 관리 계획 추천(전화/방문)
- 집중 관리 대상 리스트

## 💰 기대 효과

30일 재입원율 15%↓, 퇴원 후 관리 질 20%↑

## 📅 구현 계획

2주: 데이터 수집 + Feature / 2주: XGBoost + Survival / 1주: 위험 평가 도구 개발 / 1주: 시스템 통합
