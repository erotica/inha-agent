# 처방전 자동 검토 및 이상 알림

> 부서: 약제부

---

## 🎯 목표

NLP와 규칙 엔진으로 처방전을 자동 검토하여 용량/상호작용/이상 이상을 실시간 알림한다.

## 📊 필요 데이터

처방전 데이터(약품코드/용량/횟수/경로), 환자 진단/알레르기 정보, 약품 DB(성분/상호작용/금기), 과거 이상반응 보고, 신장/간 기능 검사 결과

## 🛠 기술 접근법

**모델:** NLP + Rule Engine + Decision Tree

**주요 Feature:**
- 약품 성분 매핑
- 최대 용량 대비 비율
- 환자 신장/간 기능
- 알레르기 코드
- 동시 처방 약품 조합
- 연령/체중
- 임신/수유 여부

## 📤 출력물

- 처방 검토 결과(PASS/WARN/STOP)
- 용량 이상 경보
- 약품 상호작용 경고
- 대체 약품 추천
- 심사 사유 설명

## 💰 기대 효과

약사 처방 검토 시간 40%↓, 조제 오류 50%↓

## 📅 구현 계획

1주: Rule Engine 구축 / 2주: NLP 처방전 파싱 / 1주: 이상 탐지 모델 / 1주: 실시간 알림 시스템
