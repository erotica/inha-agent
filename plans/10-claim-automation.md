# 진료비 심사 자동화 및 이상 탐지

> 부서: 원무팀/심사팀

---

## 🎯 목표

진료비 청구 내역을 자동 검토하여 이상 청구를 탐지하고 심사 효율을 높인다.

## 📊 필요 데이터

진료비 청구 데이터(EDI 청구), 심사 결과/조정 이력, 수가 코드 DB, 진료 기록/처방 데이터, 보험 급여 기준

## 🛠 기술 접근법

**모델:** Rule Engine + Anomaly Detection + XGBoost

**주요 Feature:**
- 청구 코드 조합
- 진단-처방-시술 일치도
- 과다 청구 지표
- 동일 코드 반복 청구
- 평균 청구액 대비 편차

## 📤 출력물

- 청구 건별 이상 점수
- 자동 심사 통과/보류 결정
- 이상 패턴 상세 분석
- 수정 추천 사항

## 💰 기대 효과

심사 시간 60%↓, 부당 청구 탐지율 40%↑

## 📅 구현 계획

1주: Rule Engine / 1주: 이상 탐지 모델 / 1주: XGBoost 분류 / 1주: 시스템 통합
