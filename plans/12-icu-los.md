# 중환자실 재실 기간(LOS) 예측

> 부서: 중환자실

---

## 🎯 목표

ICU 입실 환자의 임상 데이터로 재실 기간을 예측하여 중환자실 병상 운영을 최적화한다.

## 📊 필요 데이터

ICU 입실 기록(진단/중증도/APACHE/SOFA), 활력징후 시계열, 검사 결과 추이, 환자 기본 정보, 퇴실 경로(일반병동/퇴원/사망)

## 🛠 기술 접근법

**모델:** LSTM(시계열) + XGBoost(초기 예측)

**주요 Feature:**
- APACHE II/III 점수
- SOFA 점수 추이
- 진단 코드
- 연령/동반질환
- 첫 24h 활력징후 변동성
- 인공호흡기 사용 여부/기간

## 📤 출력물

- 개인별 예상 ICU 재실 일수(CI 포함)
- 장기 재실 고위험 플래그
- 퇴실 가능 시점 예측
- 중간 평가 Point 추천

## 💰 기대 효과

ICU 병상 계획 정확도 25%↑, 재실율 최적화

## 📅 구현 계획

2주: 시계열 데이터 전처리 / 2주: LSTM + XGBoost 앙상블 / 1주: 조기 예측 모델 / 1주: 대시보드
