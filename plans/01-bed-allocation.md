# 입원 환자 수 예측 및 병상 배치 최적화

> 부서: 병동/간호부/입원관리팀

---

## 🎯 목표

EMR 데이터를 기반으로 일별/주별 입원 환자 수를 예측하여 병상 배치를 최적화하고 병상 가동률을 향상시킨다.

## 📊 필요 데이터

과거 입원 이력(진료과, LOS, 퇴원 패턴), 응급실 입원 의뢰 건수, 외래 예약 → 입원 전환율, 계절/요일/공휴일 효과, 지역 독감/감염병 유행 데이터

## 🛠 기술 접근법

**모델:** Prophet(시계열) + Integer Linear Programming(병상 배치 최적화)

**주요 Feature:**
- 진료과별 입원율
- 요일/계절 효과
- 응급실 경유율
- 수술 예정 건수
- 퇴원 패턴
- 전년 동기 대비 증감률

## 📤 출력물

- 일별/진료과별 예측 입원 환자 수
- 추천 병상 배치표
- 과밀 주의보(가동률 90%↑)
- 병동간 환자 전과 추천

## 💰 기대 효과

병상 가동률 15%↑, 입원 대기 시간 30%↓

## 📅 구현 계획

1주: 데이터 수집 + Feature Engineering / 2주: Prophet 모델 개발 / 1주: 최적화 엔진 / 1주: 대시보드
