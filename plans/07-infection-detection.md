# 감염병 발생 조기 탐지 시스템

> 부서: 감염관리실

---

## 🎯 목표

EMR/LIS 데이터를 실시간 모니터링하여 감염병 징후를 조기에 탐지하고 확산을 차단한다.

## 📊 필요 데이터

발열 환자 발생 현황(시간/병동), 검사 결과(CRP/PCT/배양), 항생제 처방 패턴, 격리 환자 현황, 지역 감염병 유행 정보, 입원 환자 증상 기록

## 🛠 기술 접근법

**모델:** 시계열 이상 탐지 + Graph Network(확산 경로)

**주요 Feature:**
- 병동별 발열 환자 비율
- 동일 검체 양성률
- 항생제 사용량 추이
- 격리실 가동률
- 신규 감염 의심 환자 수

## 📤 출력물

- 감염 의심 조기 경보(병동/시간)
- 잠재적 감염 경로 예측
- 격리실/보호구 필요량 예측
- 역학조사용 데이터 자동 집계

## 💰 기대 효과

감염병 조기 탐지 시간 70%↓, 원내 감염률 30%↓

## 📅 구현 계획

1주: 실시간 데이터 파이프라인 / 2주: 이상 탐지 모델 / 1주: Graph Network 확산 분석 / 1주: 경보 시스템
