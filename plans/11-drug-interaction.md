# 약품 상호작용 자동 경고 시스템

> 부서: 약제부/처방

---

## 🎯 목표

환자 약물 처방 시 상호작용/금기 사항을 실시간 분석하여 안전한 처방을 지원한다.

## 📊 필요 데이터

처방전(약품코드/용량), 환자 진단/알레르기, 약품 DB(DrugBank/KIMS), 유전자 검사 정보, 신장/간 기능

## 🛠 기술 접근법

**모델:** Knowledge Graph + Graph Neural Network

**주요 Feature:**
- 약품-약품 상호작용 매트릭스
- 약품-질병 금기
- 유전자-약물 반응
- 용량-체중/신장 비율

## 📤 출력물

- 처방 시 실시간 상호작용 경고(WARN/CAUTION/STOP)
- 상호작용 기전 설명
- 대체 약품 제안
- 환자 모니터링 항목 추천

## 💰 기대 효과

약물 부작용 20%↓, 처방 안전성 30%↑

## 📅 구현 계획

2주: Knowledge Graph 구축 / 2주: GNN 모델 / 1주: 실시간 경고 시스템 / 1주: EMR 연동
