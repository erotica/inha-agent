# 의료 폐기물 발생량 예측 및 처리 최적화

> 부서: 환경관리/시설팀

---

## 🎯 목표

의료 활동 데이터를 기반으로 폐기물 발생량을 예측하여 처리 계획을 최적화하고 비용을 절감한다.

## 📊 필요 데이터

폐기물 종류별/일별 발생량(일반/감염/유해/재활용), 입원/외래/수술 환자 수, 검사 건수, 시즌/계절 요인

## 🛠 기술 접근법

**모델:** ARIMA + XGBoost

**주요 Feature:**
- 환자 수(입원/외래/수술)
- 검사 건수
- 계절/월
- 특수 폐기물 발생 행사(대규모 검진 등)
- 전년 동기 대비

## 📤 출력물

- 일별/주별 폐기물 발생량 예측
- 처리 비용 추정
- 폐기물 수거 일정 최적화
- 재활용 가능 폐기물 분리 추천

## 💰 기대 효과

폐기물 처리 비용 20%↓, 재활용률 15%↑

## 📅 구현 계획

1주: 데이터 수집 / 1주: ARIMA 예측 / 1주: XGBoost 보강 / 1주: 비용 최적화 엔진
